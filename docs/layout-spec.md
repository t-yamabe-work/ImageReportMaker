# レイアウト仕様（JSXから抽出した数値テーブル）

## 基本単位

| 項目 | 値 |
|---|---|
| ポイント→mm換算 | `1pt = 25.4 / 72 mm` |
| mm→ポイント換算 | `1mm = 72 / 25.4 pt ≒ 2.8346 pt` |

## ドキュメント

| 項目 | 値 | 備考 |
|---|---|---|
| 用紙 | A4縦 | 幅 210mm、高さは可変 |
| アートボード幅 | 210mm（≒ 595.28pt） | |
| 外枠角丸半径 | 33.47pt（≒ 11.81mm） | SVGの `path` 記述から |
| 外枠線幅 | 2.83pt | 約 1mm |

## ヘッダー領域

## 本文領域

| 項目 | 値 |
|---|---|
| 案件名（●）フォントサイズ | 22.30pt |

| 本文左端 | 8.5pt（コンテンツ左マージン3mmに相当） |

## 画像ブロック（グレー背景領域）

| 項目 | 値 | 備考 |
|---|---|---|
| 背景色 | 10%K CMYK（`#efefef`） | |
| 幅 | アートボード幅いっぱい | |
| 左右マージン（コンテンツとの差） | 3mm | `CONTENT_MARGIN_X_MM` |
| 上下マージン | 5mm | `GRAY_MARGIN_Y_MM` |
| テキスト下端からブロック上端まで | 6mm | `TEXT_TO_BLOCK_GAP_MM` |
| ブロック全高上限 | 1200mm | `MAX_HEIGHT_MM` |

## 画像グリッド

| 項目 | 値 |
|---|---|
| コンテンツ幅 | 210mm - 3mm×2 = 204mm |
| 画像間ギャップ（縦・横） | 3mm |
| 列数範囲 | 1〜5 |
| 列選択ルール | ブロック全高 ≤ 1200mm になる最小列数。満たせない場合は 5列 |
| 列幅計算 | `(204 - 3×(列数-1)) / 列数` |
| 行高 | 行内の最大画像高さに合わせる |

### 列数ごとの列幅（参考）

| 列数 | 列幅 (mm) |
|---|---|
| 1 | 204.0 |
| 2 | 100.5 |
| 3 | 66.0 |
| 4 | 48.75 |
| 5 | 38.4 |

## フォント運用

| 項目 | 選定 |
|---|---|
| 本来（Illustrator） | Kozuka Gothic Pro（Adobe Font、環境依存） |
| 本アプリ第一候補 | Hiragino Sans W3 / W6（macOS標準） |
| フォールバック | `NSFont.systemFont()`（SF Pro系） |

## 出力

| 形式 | 解像度 | 備考 |
|---|---|---|
| JPG（デフォルト） | 72dpi | A4幅 595pt × 高さ可変 |
| PNG | 72dpi | 同上 |
| SVG | ベクター | Hiragino Sans を Base64 で `@font-face` 埋め込み |

## ファイル名規則

- デフォルト：`YYMMDD.[ext]`（例：`260421.jpg`）
- カスタム名入力可。**最後に使った名前を UserDefaults に保存し、次回デフォルト表示**（A案）

## 保存先

- 初回：デスクトップ
- 2回目以降：前回指定フォルダ（UserDefaults、security-scoped bookmark で参照）
