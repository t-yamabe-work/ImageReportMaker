## 2026-04-21 プロジェクト骨組み生成

### 目的
- Illustrator + JSXスクリプト運用を置き換える macOSネイティブアプリを作る
- 初期リポジトリ構造を確立し、Core（ロジック）／App（UI）の分離を担保する

### 作成したもの
- `.gitignore` — KAKUSUベース、ビルド成果物・Xcodeユーザー設定・機密画像除外
- `README.md` — 概要・セットアップ・構成
- `Core/Package.swift` — SPMパッケージ定義（`ReportCore` モジュール）
- `Core/Sources/ReportCore/` — Models / Layout / Rendering / Persistence の骨格
- `project.yml` — xcodegen設定（`ImageReportMaker` アプリ、macOS 13+ターゲット）
- `Apps/ImageReportMaker/` — SwiftUIアプリのエントリ＋最小View
- `scripts/build-and-install.sh` — `/Applications/` 配置スクリプト
- `docs/layout-spec.md` — JSXから抽出した数値仕様表
- `docs/jsx-reference.md` — 元JSXの参照コピーと移植メモ
