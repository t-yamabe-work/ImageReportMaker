## 2026-04-21 v0.1.1 レイアウトフィードバック反映（worker1）

### 目的
- 人間運用者からのフィードバック5点を Core レイアウトに反映

### 実施内容（Core/Sources/ReportCore/ 配下のみ）
1. **`formatHeaderDate` 短縮化**（`ReportRenderer.swift`）
   - 旧: `yyyy年M月d日(EEEEE) 画像報告`
   - 新: `M/d（EEEEE）　画像報告`（例: `4/21（火）　画像報告`、全角括弧＋全角スペース）
2. **ヘッダー日付の見切れ対策**（`ReportRenderer.swift`）
   - `fittedFontSize(text:weight:baseSize:maxWidth:)` を追加
   - CTLine で実幅を計測し、右パディング込みでページ幅を超える場合のみ等比縮小
   - `PreparedLayout.dateFontSizePt` に保存し、CGContext/SVG 両方が同じ値を使用
3. **点線をグレー背景の上下端に**（`ReportRenderer.swift` / `SVGExporter.swift`）
   - ハードコード値 `138.95` を廃止
   - `dashedTopYPt = grayTopYPt`、`dashedBottomYPt = grayTopYPt + grayHeightPt` として動的算出
   - CGContext 描画 / SVG 出力 いずれも 2 本の点線を出力
4. **案件テキスト上マージン削減**（`ReportRenderer.swift`）
   - `bodyStartYPt = dashedRuleYPt + 20` → `solidRuleYPt + 15`
5. **画像ブロックへの距離短縮**（`LayoutConstants.swift`）
   - `textToBlockGapMm: 6.0 → 3.0`（SVGExporter は `prepareLayout` 結果経由で自動反映・二重描画なし）
